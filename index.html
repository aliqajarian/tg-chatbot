Telegram LLM Bot

About This Bot

This is a PHP-based Telegram bot that integrates with OpenRouter's free LLM API to provide AI-powered responses in group chats. When you mention the bot in a group chat, it will respond to your question using advanced language models.

Features

  • Responds to mentions in Telegram group chats
  • Uses free LLM models from OpenRouter
  • Easy to configure and deploy
  • Supports multiple LLM models
  • Customizable response length and behavior

Setup Instructions

  1. Get Telegram Bot Token - Talk to @BotFather on Telegram and create a new bot to get your token.
  2. Get OpenRouter API Key - Sign up at OpenRouter.ai to get your free API key.
  3. Configure the Bot - Edit config.php and add your Telegram bot token and OpenRouter API key.
  4. Deploy to Web Server - Upload all files to a PHP-enabled web server with SSL.
  5. Set Webhook - Configure Telegram to send updates to your bot using the provided URL.
  6. Add to Group Chat - Add your bot to a Telegram group chat and give it permission to read and send messages.

Usage

Once your bot is set up in a group chat, simply mention it in your message to get a response:

@yourbotname What is the capital of France?

The bot will reply directly to your message with an AI-generated answer.

Customization

You can customize the bot by modifying config.php:

  • Change the LLM model (multiple free options available)
  • Adjust the maximum response length
  • Modify the system instructions for the AI
  • Configure which chat types the bot responds in
